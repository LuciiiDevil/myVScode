This is a Student Portal/Management System developed in C# to manage student data, course enrollment, and provide a self-service portal for students. It features distinct user flows for registration, login, password recovery, and access to course information.

1. Login Section

The application starts with a selection screen, allowing users to choose between Register (for new students) and Student (for existing students to log in).

2. Student Registration

![](https://i.imgur.com/vIK3PQZ.jpeg)

The Register option leads to a detailed student registration form to capture essential demographic, contact, and parent details.

3. Portal Selection

![](https://i.imgur.com/MwlpaMq.jpeg)


The Student option directs users to the main login screen, requiring a Username and Password. It also provides links for "Forgot your password?" and "Create new account" (which links to the registration form).

4. Forgot Password / Reset

![](https://i.imgur.com/bGFMIxB.jpeg)

If a user selects "Forgot your password?", they are taken to a screen where they can request a password reset using their Email or Username.

5. Account Sign Up (Alternative Registration)

![](https://i.imgur.com/tpKBMyf.jpeg)

The "Create new account" link on the login page or a dedicated Sign Up flow allows users to create an account with a Username, Email Address, and Password.

6. Student Dashboard

![](https://i.imgur.com/IBGObR8.jpeg)

Upon successful login, the student is welcomed to their Student Portal. This dashboard clearly displays the courses they are enrolled in (e.g., Diploma in Information Technology, HND in Computing) along with a Progress bar showing their overall progress (93% in this example). It also features a navigation sidebar with links like Home, My Courses, and To Do.
